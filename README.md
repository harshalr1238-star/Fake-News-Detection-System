# Fake News Detection System

A Machine Learning and Natural Language Processing (NLP) based web application developed using Python and Django to classify news articles as Real or Fake.

## Features

* Fake News Detection using Machine Learning
* NLP-based text preprocessing
* User-friendly web interface
* Real-time news classification
* Django backend integration

## Technologies Used

* Python
* Django
* Scikit-Learn
* Pandas
* NumPy
* NLP
* HTML/CSS

## Project Structure

* detector/ - Django application
* fakenews/ - Django project settings
* train.py - Model training script
* model.pkl - Trained ML model
* tfidfvect.pkl - TF-IDF vectorizer

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run migrations

python manage.py migrate

3. Start server

python manage.py runserver

4. Open

http://127.0.0.1:8000/
